# @jtfg/sp-components

Reusable custom components and layout primitives built on top of `@fluentui/react-components`.

## Install

```bash
npm install @jtfg/sp-components
```

Peer dependencies expected in host app:
- `react`
- `react-dom`
- `@fluentui/react-components`

## Development

```bash
npm install
npm run build
npm run typecheck
```

## Exports

- `CardSection`
- `PageShell`
